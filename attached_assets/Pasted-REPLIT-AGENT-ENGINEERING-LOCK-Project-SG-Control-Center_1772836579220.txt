REPLIT AGENT ENGINEERING LOCK
Project: SG Control Center (SGCC)

Sebelum mulai build, baca file berikut sampai selesai:

SG_Control_Center_Complete_Build_Spec.md

File tersebut adalah **spesifikasi sistem final**.
Jangan mengubah struktur, jangan improvisasi, dan jangan menambahkan fitur di luar dokumen.

==================================================

ENGINEERING RULES (WAJIB)

1. Jangan over-engineer.
2. Jangan membuat arsitektur yang kompleks.
3. Jangan membuat microservices.
4. Jangan membuat terlalu banyak layer abstraction.
5. Jangan membuat fitur yang tidak diminta.
6. Jangan mengubah nama modul utama.
7. Jangan mengganti bahasa UI (harus Bahasa Indonesia).
8. Jangan mengganti tema warna (harus Navy Blue).
9. Jangan mengganti struktur role.

Fokus aplikasi ini adalah:

Executive Control Center
Monitoring
Compliance tracking
Governance dashboard

Bukan CRM.
Bukan ERP.
Bukan aplikasi chat.

DATABASE RULES

Gunakan **PostgreSQL**.

Semua tabel utama harus memiliki:

company_id

Karena sistem ini menggunakan **multi-company architecture**.

Jangan membuat database terpisah per PT.
